# Биоквиз — викторина по биологии

**Автор:** Диана Дубовая

## О проекте
Веб-приложение для изучения строения клетки. Содержит теорию, викторину из 10 вопросов, интерактивную схему и тест по картинке.

## Как запустить
1. Установи Django: `pip install django`
2. Выполни миграции: `python manage.py migrate`
3. Запусти сервер: `python manage.py runserver`
4. Открой в браузере: http://127.0.0.1:8000/

## Страницы
- Главная
- Теория
- Викторина (10 вопросов)
- Результат
- Статистика
- Интерактивная схема
- Тест по картинке

## Технологии
Python, Django, SQLite, HTML/CSS
